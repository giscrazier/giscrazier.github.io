利用PowerDesigner将表导入到数据库

  • DB
  • PowerDesigner

在利用PowerDesigner将库表设计好之后，便可以使用它将表直接导入数据库。

配置数据库连接

选中一个PDM项目，DataBase→Configure Connections..，如下图所示：


点击圈住的红色部分，Add Data Source，如下图所示：


选择“文件数据源”，点击“下一步”，如下图所示：


选择“oracle...”，因为我的数据库是Oracle，点击“下一步”，如下图所示：


输入或者浏览所要保存的位置，点击“下一步”，如下图所示：


点击“完成”。测试连接是否成功：

连接数据库

在PowerDesigner中选择DataBase→Connect，在弹出的Connect to a Data Source框中，进行配置，如下图所示：

生成数据库

接下来我们要生成数据库，在PowerDesigner中选择DataBase→Connect，选择刚才配置好要连接的数据库，如下图所示：
Generation type：
第一个是：生成sql脚本的时候勾选
第二个是：导入数据库中的时候勾选。

Generate Name in Empty Comment，勾选后数据库中就有注释了（如果设计时没有写Comment，就用Name替代），如下图所示：

这就是准备创建数据库表的sql语句，在这里浏览，如下图所示：


点击确定。生成sql脚本。脚本生成完成跳出执行的对话框，点击run即可。
